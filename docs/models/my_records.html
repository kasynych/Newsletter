My Records

This is the parent class for all model, which work with database. It performs listing, simple adding/editing etc. It's not for implementation, but for extension(like abstract)

Methods

setTable($table_name)

get($conditions)

Parameter $conditions can be either string or array or numeric
Method returns a row on base of $conditions

parent::get('subscruber_id'=>5); parent::get(5); // class gets name of primary key of self:table and compares it with '5' parent::get('subscriber.name LIKE "%Masha%"');
